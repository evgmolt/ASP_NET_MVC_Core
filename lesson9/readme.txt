Сервис отправки сообщений

Сообщения отправляются всем зарегистрированным пользователям. Управление базой пользователей осуществляется через UsersController (CRUD). Swagger интерфейс для него делать не стал, и так красивая страничка. 

Автоматическая отправка по расписанию по умолчанию включена. Включается и выключается через JobController. Расписание хранится в cron строке в файле конфигурации appsettings.json. Там же хранится текст и тема рассылаемых сообщений по расписанию, а также настройки smtp.

Ручная рассылка осуществляется через SendMessageController.
